# BUILD 051 게임 투표 페이지

게임 전시회 BUILD 051 출품작 투표 사이트입니다.

## 부문 구조

투표는 3개 부문으로 나뉩니다. 상단 탭으로 전환합니다.

- **기업 부문** — 기업 출품작 중 1표
- **학생 부문** — 학생 출품작 중 1표
- **혁신 부문** — 전체 출품작 중 1표

각 게임은 자기 소속 부문(기업/학생) 탭과 혁신 부문 탭 양쪽에 나타납니다.
관람객은 부문마다 1표씩, 총 3표를 던집니다.

### 투표 흐름

1. 탭을 옮겨가며 각 부문에서 게임의 "이 게임 선택" 버튼을 누릅니다. (부문당 1개)
2. 화면 하단 고정 바에 선택 현황과 진행도(예: 2/3)가 표시됩니다.
3. 세 부문을 모두 선택하면 "투표 제출하기" 버튼이 켜집니다. 누르면 세 부문 답이 모두
   미리 채워진 Google Form이 새 탭으로 열립니다. (덜 선택했으면 빠진 부문을 알려줍니다.)

선택 상태는 브라우저에 임시 저장되어 탭 이동·새로고침에도 유지됩니다.

## 게임 추가/수정하기

`games.json` 파일만 편집하면 됩니다. 게임 하나당 객체 하나:

```json
{
  "name": "게임 이름",
  "division": "corporate",
  "image": "images/내파일.png",
  "description": "한 줄 설명"
}
```

- `name` — **필수**.
- `division` — **필수**. `"corporate"`(기업) 또는 `"student"`(학생). 혁신 부문은 전체가 후보라 따로 지정하지 않습니다.
- `image` — 선택. 없으면 `NO IMAGE` 플레이스홀더가 표시됩니다. 이미지는 `images/` 폴더에 넣고 경로를 적습니다.
- `description` — 선택. 없으면 설명 줄 자체가 표시되지 않습니다.
- `name`은 Google Form의 게임 선택지와 **정확히 똑같이** 적어야 투표가 자동 선택됩니다.

## Google Form 연동하기

폼은 **1개**, 그 안에 **부문별 객관식 질문 3개**를 만듭니다.

1. Google Forms에서 새 폼을 만듭니다.
2. 객관식 질문 3개를 추가합니다:
   - "기업 부문 — 어떤 게임?" → 선택지 = 기업 게임 이름들
   - "학생 부문 — 어떤 게임?" → 선택지 = 학생 게임 이름들
   - "혁신 부문 — 어떤 게임?" → 선택지 = 전체 게임 이름들
   - (선택지는 games.json의 name과 정확히 동일하게)
3. 우측 상단 ⋮ → "미리 채워진 링크 가져오기" → 아무 게임이나 골라보고 "링크 가져오기".
4. 복사된 주소에서:
   - `?usp=pp_url` 까지가 `formBaseUrl` 입니다.
   - `&entry.숫자=...` 가 질문마다 하나씩 있습니다. 각 질문에 채운 값을 보고 어느 `entry.숫자`가 어느 부문 질문인지 확인합니다. (세 질문을 한 번에 미리 채워보면 entry가 3개 다 보입니다.)
5. `config.js`에 입력합니다:
   ```js
   const CONFIG = {
     formBaseUrl: "여기에 ?usp=pp_url 까지",
     entryIds: {
       corporate: "entry.기업질문번호",
       student: "entry.학생질문번호",
       innovation: "entry.혁신질문번호",
     },
   };
   ```
6. 폼 응답을 Google Sheet로 연결하면 부문별 집계가 자동으로 쌓입니다.

## 로컬에서 미리보기

```bash
python -m http.server 8000
```

브라우저에서 http://localhost:8000 접속.
(games.json을 fetch로 읽기 때문에 파일을 더블클릭으로 여는 방식은 동작하지 않을 수 있습니다.)

## 배포 (GitHub Pages)

1. GitHub에 새 저장소를 만듭니다.
2. 이 폴더의 파일을 push 합니다.
3. 저장소 Settings → Pages → Branch를 main(또는 사용 브랜치) / root로 선택 후 Save.
4. 몇 분 뒤 `https://<아이디>.github.io/<저장소명>/` 으로 접속됩니다.
5. 그 주소로 QR 코드를 만들어 전시장에 게시하세요.
